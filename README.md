# karmika_backend

Excited to share my latest personal project: Karmika HRMS! 
I’ve been building Karmika, a comprehensive Human Resource Management System designed to streamline organizational workflows, empower employees, and provide actionable analytics for HR and Admins.
Here’s a look at what the platform can do:
🌟 Key Features:
📊 Interactive Dashboards: Role-Based Access Control (RBAC) ensures Employees, Managers, HR, and Admins get a customized view of metrics that matter most to them.
📅 Leave & Attendance: Seamless check-in/check-out tracking, leave applications, manager approvals, and team attendance monitoring.
💰 Payroll & Finance: Automated salary slip generation based on dynamic salary components and structural configurations.
🎯 Performance & Appraisals: End-to-end goal setting, peer reviews, manager evaluations, and competency tracking.
🏫 Learning Management (LMS): Course enrollment, live virtual sessions, progress tracking, and automated completion certificates for employee upskilling.
🎉 Onboarding & Recruitment: Job postings, application tracking, resume validation, and structured new-hire checklists.
🤖 AI Chatbot (Karma): An integrated LLM-powered assistant to help employees navigate HR policies and answer queries instantly!
📈 System Health Monitoring: Live tracking of external API integrations and microservice health using Circuit Breakers.
🛠️ The Tech Stack:
Frontend: React.js + Vite for a blazing fast, responsive, glassmorphism-styled user interface.
Backend: Spring Boot (Java) handling robust business logic and secure API endpoints. 
Database: MySQL for reliable, relational data management. 
AI Integration: Ollama for powering the intelligent HR assistant. 
Resilience: Resilience4j Circuit Breakers to handle remote service faults gracefully.
